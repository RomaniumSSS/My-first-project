# Telegram Goal Coach Bot

> **Ваш персональный ИИ-наставник по продуктивности и достижению целей прямо в Telegram.**

## О проекте

Этот бот — не просто список задач. Это эмпатичный коуч, который помогает преодолеть страх перед большими целями, структурировать хаос и сохранять мотивацию. Он использует искусственный интеллект, чтобы превратить ваши мечты в понятный план действий.

Если вам сложно начать, вы бросаете дела на полпути или теряетесь в планировании — этот проект для вас.

##  Ключевые возможности

- ** Умный анализ целей**: Расскажите о своей мечте или загрузите мудборд — ИИ поможет сформулировать конкретные шаги.
- ** Структура и план**: Бот разбивает глобальные задачи на недели и дни, чтобы вы двигались вперед маленькими победными шагами.
- ** Поддержка и мотивация**: Если вы застряли, бот поддержит, пришлет "волшебный пинок" или совет, как справиться с прокрастинацией.
- ** Геймификация**: Получайте уровни и достижения за регулярность. Превратите саморазвитие в игру.
- ** История успеха**: Сохраняйте фото-подтверждения и создавайте ленту своих достижений ("мемуары успеха").

##  Как запустить

Проект упакован в контейнеры для быстрого старта.

1. **Клонируйте репозиторий**:
   ```bash
   git clone <url-репозитория>
   ```
2. **Настройте окружение**:
   Создайте файл `.env` (см. пример) и укажите свои API ключи (Telegram, OpenAI).
3. **Запустите через Docker**:
   ```bash
   docker compose up -d
   ```
